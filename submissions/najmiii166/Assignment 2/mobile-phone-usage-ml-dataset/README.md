# Mobile Phone Usage and Battery Drain Dataset

This repository contains an original dataset created for a Machine Learning assignment on Data Foundations.

## Dataset Overview
- Samples: 50 rows
- Features: 6
- Label: 1
- Total Columns: 7
- Data Source: Self-created (manual, simulated survey-based data)

## Files
- `data/mobile_phone_usage.csv` – the dataset
- `paper/dataset_description.md` – short research paper describing the dataset

## Intended Use
This dataset is designed for educational purposes, especially for:
- Data preprocessing
- Feature engineering
- Classification, regression, and clustering tasks

No external datasets (Kaggle, UCI, etc.) were used.
